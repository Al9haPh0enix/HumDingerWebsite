Hum-Dinger Design:
The 'Wooden Hub' is put on the 'Medium Stick' so that it rotates. A 'Paperclip' is attached to the 'Motor's positive and to the 'Wooden Hub' so that it rotates with the 'Wooden Hub'. The 'Paperclip' is then pushed into another 'Paperclip' which is attached to the 'Battery's positive. The 'Motor's negative is then attached to the 'Battery's negative.
